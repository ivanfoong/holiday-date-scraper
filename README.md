## Usage

### HolidayAPI
1. Signup for an account at <https://holidayapi.com/>
2. Login and find your `API_KEY`
3. Use the follow command (Remember to replace `<API_KEY>` with your `API_KEY`)
```
HOLIDAY_API_KEY=<API_KEY> python3 scrape_holidayapi.py
```
